PONTEVEDRA´S THEATRE

Introduction

This application was born as an school project at the University of Santiago de Compostela (USC, Spain) as JavaFX Application to sell and views tickets for the Pontevedra´s Theatre

The guidelines given by the teacher were to create an App based on a ticketing touchpad for a Theatre, using UX techniques as Mental maps, prototyping, Nielsen´s Heuristics and questionnaires with users

You can check it out at my GitHub page for more information
